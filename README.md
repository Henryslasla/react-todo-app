# React Todo App

A simple Todo application built using React.

This project demonstrates core React concepts like state management, conditional rendering, and side effects using useEffect.

## Features
- Add and delete todos
- Mark tasks as completed
- Filter (All / Active / Completed)
- Data persistence using localStorage

## Tech Stack
- React (Hooks)
- JavaScript
- HTML/CSS

## How to Run
npm install
npm run dev
